Desarrollos a medida

Ruby on Rails

Cuando un CMS no es suficiente, desarrollamos aplicaciones a medida. Para ello utilizamos Ruby on Rails

t-engine

T-engine es una plataforma de e-learning adaptada a cursos para el sector farmacéutico y sanitario impartidos en Hospitales y Centros de Salud de todo el territorio español.

  • Cliente: Think-in Pharma Services SL
  • Aplicación a medida (Ruby on Rails)
  • Desarollo inicial: 5 semanas.
  • Desarrollo de cada nuevo curso: 2 semanas.

Ejemplos:

  • talleresanemia.es
  • cuidadosenfermeria.org

Storemonitor

Storemonitor es una aplicación de gestión de fotografías de tiendas para el control del surtido.

  • Cliente: Colgate-Palmolive Iberia (ES + PT)
  • Aplicación a medida (Ruby on Rails)
  • 9000 tiendas en España y Portugal
  • 4000 fotografías nuevas cada mes
  • Importación automatizada desde otros proveedores
  • Búsqueda y filtrado de fotografías
  • Generación de informes de seguimiento.

OHWR

El Open Hardware Repository es una aplicación de gestión de proyectos.

  • Cliente: CERN
  • Software: Redmine
  • Forums, Wikis, Tickets e incidencias
  • Desarrollo de plug-ins a medida
  • Repositorios git y subversion
