h3. Aplicaciones web

bq. "Te envío este excel para que lo revises."

Nuestra filosofía es: todo el mundo usa la web. Es un medio universal de comunicación.

Utilizar una página web para controlar un flujo de trabajo:

* Facilita la compartición de datos entre usuarios
* Deslocaliza la información
* Añade seguridad

h3. Aplicaciones fáciles de usar

bq. "¿Dónde tengo que hacer click?"

bq. "Oups! He vuelto a borrar mis datos"

* Un botón para cada cosa
* No más botones de los necesarios
* Confirme siempre antes de borrar cualquier cosa


h3. Aplicaciones útiles

bq. "No es lo que pido, sino lo que quiero."

Analizamos el problema y las expectativas del cliente y:

* Desarrollamos una aplicación ajustada a sus necesidades o bien
* Encontramos la solución ya existente que mejor se ajuste
